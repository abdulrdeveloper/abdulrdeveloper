## ✦ Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/abdulrdeveloper/abdulrdeveloper.git
cd abdulrdeveloper
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the dev server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```

## ✦ Customize It — Make It Yours

Here's where to update your own info:

| What | Where |
|------|-------|
| Your name, title, bio | `src/components/Hero.jsx` (or equivalent) |
| Your skills / tech stack | `src/data/skills.js` (or equivalent) |
| Your projects | `src/data/projects.js` (or equivalent) |
| Social links | `src/data/links.js` (or equivalent) |
| SEO title, description | `index.html` |
| OG image | `public/og-image.png` — replace with your own 1200×630px image |
| Favicon | `public/favicon.png` |
| Color theme | `src/index.css or App.css` |

> **Tip:** Search for `"Abdul Rahman"` across the project — replace every instance with your own name.
